### Setup

1. Clone the repository and navigate to the project folder:
   ```bash
   git clone <repository-url>
   cd types-hero-level-sorter
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Compile TypeScript:
   ```bash
   npx tsc
   ```

4. Run the program:
   ```bash
   node index.js
   ```
